PLACEHOLDER ASSETS NEEDED

Before running the app, you need to add the following image files to this directory:

1. icon.png (1024x1024 px)
   - This is your app icon that appears on the home screen
   - Should contain your church logo or a representative image
   - Use a square design with no transparency

2. splash.png (1242x2436 px for iPhone X)
   - This is the splash screen shown when the app launches
   - Can include church name, logo, or welcoming image
   - Recommended size ensures compatibility with all devices

3. images/church-logo.png (optional)
   - Additional church logo for use within the app
   - Any reasonable size (recommend 512x512 px)

HOW TO CREATE THESE ASSETS:

Option 1 - Quick Placeholders:
- Go to https://placehold.co/1024x1024/png for icon
- Go to https://placehold.co/1242x2436/png for splash
- Download and rename accordingly

Option 2 - Custom Design (Recommended):
- Use Canva (https://canva.com) - free templates available
- Use Figma (https://figma.com) - professional design tool
- Use Photoshop or other design software
- Hire a designer on Fiverr or similar platforms

Option 3 - Generate with AI:
- Use DALL-E, Midjourney, or similar AI image generators
- Provide prompts like "church building icon, simple, clean design"
- Export at the required sizes

IMPORTANT:
The app will not build without these assets in place. However, you can test
the app using Expo Go during development without proper assets (Expo will use
default placeholders).
